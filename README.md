# 🎮 Tic-Tac-Toe Game

A modern Tic-Tac-Toe game built with JavaScript, featuring multiple difficulty levels, interactive UI, and smart AI.

---

## 🚀 Features

* 🧑‍💻 Single-player mode with difficulty levels:

  * Easy (random moves)
  * Medium (blocks player moves)
  * Hard (Minimax algorithm — unbeatable)
* 🧮 Score tracking system
* 🎨 Clean and interactive UI
* ✨ Animations for moves and results
* 🌗 Custom themes (light/dark mode)
* 🔊 Sound effects (click, win, draw)
* 📱 Fully responsive (mobile, tablet, desktop)
* ⏱️ Turn timer (optional feature)

---

## 🧠 Game Logic

* 3x3 grid system
* Two players: X and O
* Turn-based gameplay
* Win detection:

  * Rows
  * Columns
  * Diagonals
* Draw detection when the board is full

---

## 🤖 AI (Single Player Mode)

* **Easy**: Random move selection
* **Medium**: Blocks opponent’s winning moves
* **Hard**: Uses the **Minimax algorithm** to play optimally (unbeatable)

## 🎮 How to Play

1. Select game mode (solo or multiplayer)
2. Choose difficulty level
3. Click on a cell to make a move
4. Align 3 symbols to win!
